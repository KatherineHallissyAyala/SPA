Computer Class Curriculum

What goals can teachers set in a K-12 computer course?

Teach Scratch, Python, or HTML If your school dedicates time in the schedule to computer classes, use that time to give students a taste of computer programming. Scratch is a great tool to teach programming at a young age with graphic icons for each command. This prevents the frustration associated with typo bugs, and it makes the concepts very visual. In middle or high school, students should be ready for a typed language like python.

Develop a Mindset for Identifying and Extending Patterns To do this, students need time to explore a few software interfaces at a deep level. Animation involves learning to draw using a computer, and then learning an animation tool.

Start a Podcast Technology offers an outlet for self-expression. Kids have a lot to say. Invite kids to write, interview, and record. Teach audio editing tools. Expose students to podcast publishing tools.
